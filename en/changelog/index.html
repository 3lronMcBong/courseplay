    Changes in version 3.20

    • Conversion to FS 2013
    • Support for trailer with multiple tipping sides
    • Automatic horizontal offset (Combi mode)
    • Pathfinding deactived (there were too many problems)
    • Support for beet and potatoe harvesters
    • Optimized turning maneuver in combi mode
    • Value (speeds, percentages etc.) in the hud can be altered in bigger steps when pressing the CTRL key.
    • Debug-Level for better debugging information
    • Unloading in a BGA silo while driving
    • Removed double use of INS key
    • "In traffic" info message is displayed again
    • Keyboard keys for using Courseplay changed to CTRL+KP 7, CTRL+KP 8 and CTRL+KP 9 (instead of not using the CTRL key)
    • manuelles reverse driving: In order to record reverse driving, the driving direction must be switched in the hud
    • many more bugfixes
    • many more bugs
