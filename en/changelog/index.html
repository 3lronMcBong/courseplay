    • Changelog

    Changes in version 3.30

    fix
    The never ending story of improving the automatic offset - should work with all standard machinery, including attachable combines
    new

    BGA: division into 3 segments during the unloading process

    The silo is divided into three segments, and the tractor (driver) automatically decides which section to unload into, based on which is the emptiest. After a couple of trailers have been unloaded, this should guarantee an even distribution in the silo
    new

    Speeds

    • The tractors should now adapt better to the combine's/chopper's speed during unloading (combi mode)
    • The speed settings now support the ESLimiter (version 3 and up) – quite beautifully if we may say so
    • new speed setting "Unloading (BGA)" for, surprise surprise, the speed during unloading in the BGA silos
    • Speed adaption to traffic flow
    fix
    Vast improvements with the turn maneuvers during unloading the chopper (combi mode). Tractors crashing into the chopper should never happen at all anymore. Now even the turn radius gets a second life, it will be important to set it fitting to the current tractor
    change
    new turn maneuver for unloading a combine (combi and overloader mode)
    fix
    Small improvements for the forced driver side during unloading the chopper (combi mode)
    fix
    The driver should now stop at the first waypoint in "Driver" mode
    fix
    Automatic refueling should work with most machines again
    fix
    The course list can be scrolled through faster by using the CTRL key
    fix
    improved calculation which driver should head to a calling combine (based on distance)
    fix
    fixed graphics warnings (2^n), incl. the display of the yellow arrow when pausing the recording
    fix
    Forage wagons will be re-activated upon resuming the field course (lowering and activating the pickup)
    new
    Display of speed in mph for English game versions
    new

    Automatic calculation of turning circle

    • based on wheel base, wheel track, steering angle and amount of trailers
    • set from 'auto' to 'manual' with +/-
    • set from 'manual' to 'auto' by setting it to 0 (zero)
    fix
    Three sided tippers (unloading left or right) are able again to unload (and continue to work afterwards)
    new

    Seeding mode

    • "Fertilizer" mode has become "Fertilizer and seeding"
    • automatic (un)folding, raising/lowering, (de)activating
    • like fertilizers, empty seeders can be refilled automatically by driving through a seed trigger
    new
    Field work mode supports new tools: cultivators, ploughs, windrowers and tedders
    fix
    Patch 1.4: Yellow arrow won't be shown during loading the map
    new
    Fertilizer mode: support for Fuchs liquid manure barrel and Tebbe HS180 manure spreader
    fix
    Support for horizontal offset in fertilizer/seeding mode
    change

    Completely new "Combine drives to trailer and unloads" mode

    • The combine returns and aligns with its lane way more accurately
    • Pipe is opened earlier
    • AI helper course and unloading course aren't mixed together and saved anymore
    change
    Work width is shown as a red line (better visibility), including horizontal offset
    change
    Course is followed 50% more accurately – waypoints are skipped less frequently
    new
    Support for Manual Ignition
    new

    Courseplay en fran√ßais

    In French game versions, the Courseplay hud is shown in … french.
    new

    Combine in fieldwork mode

    • automatic starting and stoping the cutter
    • automatic pipe opening/closing
    • Waiting for unloading driver
    change
    Fieldwork and fertilizer/seeding modes: in order to prevent holes in the field, tools are reactivated earlier upon returning to the point where work was aborted
    fix
    Fertilizer/seeding mode: possible crash when waiting points were missing has been fixed
    new
    Fieldwork mode: support for multiple tools at the same time (e.g. front/rear mowers; moher and forage wagon etc.)
    new
    Trailers with a dynamic cover open and close it automatically, based on the driving situation. Currently only Eifok's Agroliner SMK34 is supported.
    bug
    Game crashes when a course has more than two waiting points in fertilizer/seeding mode
    bug
    Course generation: when the field edge course starts or ends at a field corner, the field course isn't generated correctly (all lanes will go through the field's center point, resulting in an "X" course)
    bug
    attached combines aren't supported in fieldwork mode
